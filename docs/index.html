VESTA — Detecção de Quedas para Idosos

Sistema inteligente de detecção de quedas para idosos, composto por dispositivo vestível conectado via Wi-Fi a um aplicativo mobile. O sistema identifica automaticamente quedas e envia alertas imediatos por SMS para familiares ou cuidadores.

Minha participação: Atuei como Product Owner do projeto, responsável pela definição de requisitos, priorização do backlog, coordenação da equipe de desenvolvimento e garantia de que o produto atenda às necessidades dos usuários. Participo ativamente das cerimônias Scrum e na tomada de decisões estratégicas do projeto.
React Native TypeScript ESP32 Node.js MongoDB MQTT
